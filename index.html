Use Case: Wunschliste

    im OnlineShop werden unterschiedliche Produkte verkauft
    die Produkte haben eine ID und einen Namen
    Kunden können sich alle Produkte anzeigen lassen
    Kunden können Produkte über die ID oder den Namen suchen
    Produkte können Wunschlisten zugeordnet werden
    Wunschlisten enthalten eine ID sowie einen Titel und sind einem Kunden zugeordnet
    Kunden können mehrere Wunschlisten für sich selbst erzeugen
    Kunden können Wunschlisten löschen
    Kunden können Wunschlisten Produkte hinzufügen
    Kunden können von Wunschlisten Produkte entfernen
    Kunden können neue Produkte für die Wunschlisten erstellen oder bestehende löschen
    Kunden können nur über ihre ID gefunden werden, haben jedoch auch einen vollständigen Namen
